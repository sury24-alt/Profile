# **App Name**: Surya Teja Portfolio

## Core Features:

- Homepage Introduction: Display a brief personal bio and a professional profile picture.
- About Me Section: Detail educational background, current involvements (SIH hardware projects, web development internship), and future interests (machine learning, UI/UX design).
- Skills Showcase: Visually represent skills like Python, HTML, CSS, Java, hardware projects, and machine learning basics using interactive elements such as skill bars or animated icons.
- Services Offered: Outline services such as web design, Python project development, and portfolio creation, each with descriptions and icons.
- Portfolio/Projects Display: List current and future projects, including SIH hardware project and web development internship work. Include a 'Coming Soon' section for projects in development.
- Contact Information: Provide contact information, including email (suryapedapudi2404@gmail.com) and GitHub (sury24-alt), with an option for a simple contact form.

## Style Guidelines:

- Primary color: Shades of blue (#4682B4) to represent innovation, confidence, and stability.
- Background color: Very light blue (#E6F0FF), almost white, for a clean and unobtrusive background.
- Accent color: A slightly purple shade of blue (#6A5ACD) will provide a high-tech, cutting-edge look.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-focused headline.
- Body font: 'Inter' (sans-serif) for a clean and readable body text.
- Use a set of minimalist icons, related to development, AI, and design. Color the icons with the accent color.
- Clean, minimalist layout with intuitive navigation. Responsive design for mobile and desktop.
- Subtle fade-in animations for sections, text, and images.
